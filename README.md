# apbd-cw3-equipment-rental
# Equipment Rental System

## Opis
Aplikacja konsolowa w C# do zarządzania wypożyczalnią sprzętu.
Obsługuje użytkowników, sprzęt oraz proces wypożyczania i zwrotów.

## Funkcjonalności
- Dodawanie sprzętu i użytkowników
- Wypożyczanie i zwracanie sprzętu
- Ograniczenia wypożyczeń:
    - Student: max 2
    - Employee: max 5
- Obsługa błędów (np. przekroczenie limitu)

## Struktura projektu
- Models – klasy danych (Equipment, User, Rental)
- Services – logika biznesowa
- Enums – statusy sprzętu

## Uruchomienie
Uruchom projekt w Riderze lub Visual Studio
